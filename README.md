# Triangulate-AU_01-Armband-Interface

## Description

A sci-fi-style armband interface running Android 14 on a Rockchip RK3576 module in a custom carrier board. Features an 8-inch 2.8K curved AMOLED with touch capabilities, a 2.1-channel speaker system with a 20W subwoofer, a 1080p pop-up webcam, various sensors, a huge battery at approx. 80Wh and 65W USB-PD charging. Perfect for use in the field with its bright display and long battery life.

## Enclosure

### Features
* 185mm long, with a radius of 56mm in the display section
* Four battery compartments housing 16x 1200mAh batteries with integrated cable channels
* Upwards-facing speaker channels for mid-high frequency drivers
* Integrated sealed subwoofer chamber
* Slim 38mm wide compute board in a hinged section
* Contoured inner profile (in contact with arm)
* Pop-up laptop webcam

## Electronics

### Features
* PCIe x1 slot for NVMe SSD
* HDMI 2.1 port for display
* Two USB-C ports
* Port 1: USB-2.0, USB-PD-3.1 sink (Max. 20V) (for battery charging)
* Port 2: USB-3.0 SuperSpeed, USB-2.0, 5V 3A BC1.2 source (for peripherals)
* Approx. 60W battery charging via BQ25798
* Audio digital signal processing via ADAU1701
* 20W PAM8320 amplifier (for subwoofer)
* 2x 2W PAM8302 amplifiers (for L/R channels)

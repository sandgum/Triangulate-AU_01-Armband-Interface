# Triangulate-AU_01-Armband-Interface
A sci-fi-style armband interface running Android 14 on a Rockchip RK3576 module in a custom carrier board. Features an 8-inch 2.8K curved AMOLED with touch capabilities, a stereo speaker system with subwoofer, a webcam, various sensors and a huge battery at approx. 50Wh.
